# Building an Angular Compiler with Google Gemini

This is an exploration in building an Angular Compiler "lite" version.

Blog post: https://dev.to/brandontroberts/building-an-angular-compiler-with-google-gemini-ie0

## Setup

Run `npm install` to install the application dependencies.

## Development

Run `npm start` for a dev server. Navigate to `http://localhost:5173/`. The application automatically reloads if you change any of the source files.
